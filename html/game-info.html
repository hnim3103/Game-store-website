We need Uncharted 1–3

I think Uncharted 4 and Lost Legacy's biggest weakness isn't about the games themselves, but Sony's unwillingness to port the first three games to PC...

I think Uncharted 4 and Lost Legacy's biggest weakness isn't about the games themselves, but Sony's unwillingness to port the first three games to PC. I never played them because I never owned Playstation after 2, but I heard these games are self-contained, and while that is somewhat true for the main adventure itself, and the secondary main character, Nathan's brother Sam, as well as the villains are first introduced in this game as I understand, there's still a lot of history with the characters that you just don't get from just these games. The games themselves are still excellent though. Gunplay is maybe the weakest aspect but the combat is still fun, and a lot of encounters can be done entirely in stealth.
